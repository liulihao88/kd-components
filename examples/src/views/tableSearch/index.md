### 事件
| 事件名称     | 说明      | 回调参数 |
|----------|---------|------|
| onFold   | 折叠或展开回调 |      |
| onSearch | 点击搜索回调  |      |
| onReset  | 点击重置回调  |      |

### 插槽
| 名称    | 说明         | 回调参数 |
|-------|------------|------|
| -     | 首行左侧筛选条件区域 |      |
| right | 首行右侧操作按钮区域 |      |
| more  | 高级搜索对应下拉区域 |      |
